How to use this app

Recording a discovery

Go to the Crafting tab, pick the two materials you used, and click Show Combinations. Find the code that matched and click Record discovery — the form opens pre-filled with the materials and code. Give the item a name and its card number, then save.

You can also add a recipe manually from any tab using + Add Recipe in the header.

Crafting codes

Codes can be typed in shorthand: one letter for the colour followed by four digits — e.g. R1311 for Red 1311. Letters: B = Blue, R = Red, Y = Yellow, P = Purple, G = Grey, Gn = Green, O = Orange. Separate multiple codes with commas or spaces.

Crafting tab card colours

  • Green — item already discovered with this combination
  • Terracotta — tried this code, nothing crafted
  • Beige — not yet tried

Explorer filters

All — shows every computed code for the selected materials.
Unknown — hides combinations that have already been resolved (discovered or dead-end).
Known — shows only material pairs where at least one item has been found.

Sharing with the table

Use Export JSON to save your current data, then share the file. Others import it with Import JSON — choose Merge to combine with their existing data, or Overwrite to replace it entirely.
